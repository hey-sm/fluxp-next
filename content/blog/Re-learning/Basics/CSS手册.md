placeholder
```css
/* 修改placeholder文字颜色 */
input::placeholder,
textarea::placeholder {
  color: #999; /* 设置你想要的颜色 */
}

/* 修改placeholder文字大小 */
input::placeholder,
textarea::placeholder {
  font-size: 14px; /* 设置你想要的字体大小 */
}
```
